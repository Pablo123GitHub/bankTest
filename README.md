# Individual technical challenges


### Task

This repository contains a technical test in JavaScript for a bank

### Acceptance criteria

Given a client makes a deposit of 1000 on 10-01-2012
And a deposit of 2000 on 13-01-2012
And a withdrawal of 500 on 14-01-2012
When she prints her bank statement
Then she would see

```
date || credit || debit || balance
14/01/2012 || || 500.00 || 2500.00
13/01/2012 || 2000.00 || || 3000.00
10/01/2012 || 1000.00 || || 1000.00
```

### Testing framework
I am using JASMINE to unit test the code. In order to run the test open spec/frontend/SpecRunner.html on a browser.
